========================

⚡ 快速导航
========================

通常需要编辑的文件：
✏️  docs/*/xxx.md       添加或修改笔记
✏️  docs/.vitepress/config.mts   修改网站配置

一般不需要修改的文件：
❌ .github/workflows/deploy.yml   (GitHub Actions 自动部署)
❌ package.json                   (NPM 依赖配置)
❌ node_modules/                  (NPM 依赖文件)

========================

📖 文件用途速查
========================

|  文件  |  用途  |
|--------|--------|
| README.md | 项目总览、快速开始、更新日志 |
| DEPLOY.md | GitHub Pages 部署详细指南 |
| QUICKSTART.md | 常用命令、故障排查、快速操作 |
| DEPLOYMENT_SUMMARY.md | 部署总结、后续步骤 |
| docs/index.md | 网站首页内容 |
| docs/*/*.md | 各个主题下的笔记页面 |
| docs/.vitepress/config.mts | VitePress 网站配置 |
| .github/workflows/deploy.yml | GitHub Actions 自动部署脚本 |

========================

🚀 立即开始 3 步
========================

1️⃣  启动开发服务器
   $ npm run docs:dev
   → 浏览器访问 http://localhost:5173

2️⃣  编辑笔记
   编辑 docs/ 目录下的 .md 文件
   实时预览更改

3️⃣  推送到 GitHub
   $ git add .
   $ git commit -m "Update notes"
   $ git push origin main
   → 自动构建和发布
