# C++基础与算法总结

这个页面先作为复习总表，目标不是覆盖所有知识点，而是把最常用、最容易混淆、最值得反复看的内容压缩成一页。

## C++基础总结

### 1. 变量、类型与作用域

- 基本类型：`int`、`long long`、`double`、`char`、`bool`
- 常量定义优先用 `const` 或 `constexpr`
- 局部变量作用域最清晰，尽量少用全局变量
- 注意整数溢出，算法题里涉及范围时优先判断是否需要 `long long`

### 2. 引用、指针与值传递

- 引用是已有对象的别名，语法更安全，常用于参数传递
- 指针保存地址，适合动态内存、链表、树等结构
- 大对象传参优先 `const T&`
- 需要修改实参时可用 `T&`

### 3. 数组、字符串与容器

- `vector`：最常用动态数组
- `string`：字符串处理主力
- `map` / `unordered_map`：做映射
- `set` / `unordered_set`：去重与查找
- `queue` / `stack` / `priority_queue`：BFS、单调结构、堆

### 4. 容易出错的点

- 数组越界
- 野指针
- 变量未初始化
- 传值导致性能损失
- 递归爆栈
- `=` 和 `==` 混用

## 算法总结

### 1. 时间复杂度基本观念

- `O(1)`：常数级
- `O(log n)`：二分
- `O(n)`：单次遍历
- `O(n log n)`：排序、分治
- `O(n^2)`：双层循环

### 2. 高频方法

- 排序
- 二分
- 双指针
- 前缀和与差分
- 哈希
- DFS / BFS
- 动态规划
- 贪心
- 回溯

### 3. 刷题时的标准思考顺序

1. 题目输入规模多大？
2. 暴力做法是什么？
3. 哪里重复计算了？
4. 能不能排序、哈希、前缀和、二分、双指针优化？
5. 这题本质是搜索、图、DP 还是贪心？
